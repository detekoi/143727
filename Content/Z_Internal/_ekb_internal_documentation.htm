class="title">linking to the web stylesheet

this design inherits some basic css from the site using a file common.css

  1.  On the "internal" target
    1. set "master style sheet" to common.css which is a local copy of the web common.css (from git). (done once per new target)
    2.  common.css must link to private.css : open common.css, from options select stylesheet links select private.css - done every time common.css is updated (or just add @import url('private.css') ; zg the top of the common.css style sheet)
  2. On the"public" target

    1. in contente/masterpages/<all maser pages> add link to web style

      <link type="text/css" rel="stylesheet" href=""/>

    2. set "master style sheet" to private.css

File naming

All file names should use lowercase and "-" and not "_" or spaces.

File names should be names with a leading "_" until they are after review - then the "_" should be removed (this makes it easy to ensure they are not pushed to production by mistake)
