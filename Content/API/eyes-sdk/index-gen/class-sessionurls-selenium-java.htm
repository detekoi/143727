SessionUrls class

Platform: Selenium Language: Java
other languages
An object of this class is returned as part of the information in the lookupid class. This class provides URIs that can be used in a browser to display the Test manager batch and test results for this test.
getBatch()
Return a URI, that when opened in a browser displays the Applitools Test manager test results page, loaded with the batch of this test.
getSession()
Return a URI, that when opened in a browser will display the Applitools Test manager test results page loaded with the results of this test.
