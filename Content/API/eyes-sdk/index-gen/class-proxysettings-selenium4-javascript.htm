ProxySettings class

Platform: Selenium 4 Language: JavaScript
other languages
An object used to set and retrieve the details of the proxy server used to interact with the Eyes server.
To set a proxy server, create an object of this type and pass it to the lookupid method.
ProxySettings()
This is the constructor for the lookupid class.
getPassword()
Return the proxy password set when the object was created.
getUri()
Return the proxy URI set when the object was created.
getUsername()
Return the proxy username set when the object was created.
