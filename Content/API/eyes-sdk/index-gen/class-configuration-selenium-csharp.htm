Configuration class

Platform: Selenium Language: C#
other languages
This class is used to create a configuration object that is passed to lookupid.
You can use this configuration object instead of using the various methods and properties that set the same ttributes.
AgentId
This property can be used to set and retrieve the name and version of the SDK.
AppName
This property can be used to set and retrieve the value of the application name property.
BaselineBranchName
This property can be used to set and retrieve the name of the branch where the baseline reference will be taken from and where new and accepted steps will be saved to.
BaselineEnvName
This property can be used to set and retrieve the name of the environment that will be used to determine the baseline.
Batch
This property can be used to set and retrieve the batch for this test.
BranchName
This property can be used to set and retrieve the branch used for the baseline of this run.
ForceFullPageScreenshot
This property can be used to set and retrieve whether or not Eyes does scrolling and stitching on scrolled pages.
HideCaret
This property can be used to set and retrieve if Eyes hides the caret before capturing the screenshot.
HideScrollbars
This property can be used to set and retrieve whether or not Eyes hides the scrollbars before capturing screenshots.
ParentBranchName
This property can be used to set and retrieve the parent branch, from which newly created branches get their initial baseline.
SaveDiffs
This property can be used to set and retrieve if steps that have mismatches should be automatically saved to the baseline or not.
StitchMode
This property can be used to set and retrieve the type of stitching used for full page screenshots.
TestName
This property can be used to set and retrieve the name of the test.
ViewportSize
This property can be used to set and retrieve the viewport size to be used by the test.
WaitBeforeScreenshots
This property can be used to set and retrieve the amount of time Eyes waits before capturing a screenshot.
