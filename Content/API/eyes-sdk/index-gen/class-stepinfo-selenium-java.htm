StepInfo class

Platform: Selenium Language: Java
other languages
When a test terminates, the lookupid method returns an object with methods/properties defined by the lookupid class. Within the lookupid there is an array of objects which have the properties defined by the class lookupid. Each element of this array describes information for one step in the test.The properties in this class are used to retrieve information for a given step in the test results.
You can access the array of objects with the step information using the lookupid
HasBaselineImage()
Returns whether the baseline has a baseline image for this step.
getName()
Returns the name (tag) assigned to the step (in the check method).
isDifferent()
Use this method to if the checkpoint did not match a baseline image.
