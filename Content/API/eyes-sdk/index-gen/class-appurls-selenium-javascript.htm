An object of this class is contained in the lookupid object which is in itself an object in the lookupid object returned by the lookupid method.The properties of this object are used to retrieve information about each of the steps in the test results. Although there is no named class for this object, for convenience it is documented in a class like style - see lookupid.
step
The value of this property is a URI, that when opened in a browser will display the Test Manager Test result page scrolled to the thumbnail for this step.
stepEditor
The value of this property is a URI, that when opened in a browser will display the Test Manager Step Editor page loaded with the results of this step.
