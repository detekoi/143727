Eyes class

Platform: Selenium Language: JavaScript
other languages
This class provides the main methods used to execute tests and checkpoints.
abortIfNotClosed()
Call this in an exception handler that encloses the entire test to catch cases where lookupid was called, an exception is thrown, and execution stops without calling lookupid.
addProperty()
Adds a custom key name/value property that will be associated with tests. You can view these properties and filter and group by these properties in the Test Manager.
check()
Run a checkpoint. Uses Fluent arguments to specify the various parameters.
checkElement()
Run a checkpoint on the area of a particular element.
checkElementBy()
Run a checkpoint only elements identified by a locator.
checkFrame()
Run a checkpoint on the area of a particular frame.
checkRegion()
Run a checkpoint only in a particular region of the window.
checkRegionBy()
Run a checkpoint only on a specific region in the window identified by a locator.
checkRegionByElement()
Run a checkpoint only on a specific region defined by an element.
checkWindow()
Runs a checkpoint of the entire application (e.g. browser) window.
close()
Call this method at the end of the test (after all the checkpoints have been called).
eyes()
This is the constructor for the lookupid class.
getAgentId()
Use this method to retrieve the name and version of the SDK.
getApiKey()
Use this method to retrieve your Eyes license key that you set with lookupid, or was extracted from the APPLITOOLS_API_KEY environment variable.
getAppName()
Use this method to retrieve the name of the application property.
getBaselineBranchName()
Use this method to retrieve the name of the branch where the baseline reference will be taken from and where new and accepted steps will be saved to.
getBaselineEnvName()
Returns the name of the baseline environment defined for this test run.
getBatch()
Use this method to retrieve the batch information for tests.
getBranchName()
Use this method to retrieve the branch used for the baseline of this run.
getDefaultMatchTimeout()
Use this method to retrieve the maximum time Eyes will try to perform a match on the fully captured image.
getForceFullPageScreenshot()
Use this method to retrieve whether or not Eyes does scrolling and stitching on pages that are larger than the viewport.
getForcedImageRotation()
Use this method to retrieve the rotation applied to checkpoint images before matching.
getHideScrollbars()
Use this method to retrieve whether or not Eyes hides scrollbars before capturing screenshots.
getHostOS()
Use this method to retrieve the name that is used for the Host OS property.
getHostingApp()
Use this method to retrieve the name that is being used for the host application (browser) property.
getIgnoreCaret()
Use this method to retrieve whether or not Eyes eliminates mismatches due to blinking cursor artifacts.
getIsDisabled()
Use this method to retrieve whether or not interactions with Eyes is being ignored.
getIsOpen()
Use this to know if the Eyes instance is currently open
getParentBranchName()
Use this method to retrieve the parent branch, from which newly created branches get their initial baseline.
getProxy()
Use this method to retrieve the proxy settings used for the proxy.
getSaveDebugScreenshots()
Use this method to retrieve whether or not checkpoint images are copied to a local file (for debugging).
getSaveFailedTests()
Use this method to retrieve whether or not failed tests are saved by default
getSaveNewTests()
Use this method to retrieve whether or not 'new' tests are saved to the baseline by default.
getScaleRatio()
Use this method to retrieve the scale ratio that is applied to images before matching.
getScrollRootElement()
Returns the element to which lookupid will be applied
getServerUrl()
Use this method to retrieve the URL of the Eyes server used to process the captured images.
getStitchMode()
Use this method to retrieve the type of stitching used for full page screenshots.
getViewportSize()
Use this method to retrieve the current viewport size.
getWaitBeforeScreenshots()
Use this method to retrieve the amount of time Eyes waits before capturing an image.
open()
Call this method to start a test, before calling any of the check methods.
replaceWindow()
Replace the captured image in a step with an image provided as a parameter.

Use this method to set the name and version of the SDK.
