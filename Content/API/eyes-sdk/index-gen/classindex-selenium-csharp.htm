ApiUrls
An object of this class is contained in the lookupid object which is in itself an object in the lookupid object returned by the lookupid method.The methods in this class are used to retrieve a URI that can be used with the REST API to obtain information about a particular step.
AppUrls
An object of this class is contained in the lookupid object which is in itself an object in the lookupid object returned by the lookupid method.The properties in this class are used to retrieve a URI that can be used in a browser to open the Test Manager on a page that displays information about a particular step.
BatchInfo
Call lookupid with an object of this class to cofigure the batch for one or more tests.
Configuration
This class is used to create a configuration object that is passed to lookupid.
Eyes
This class provides the main methods used to execute tests and checkpoints.
SeleniumCheckSettings
The methods in this class are used as part of the lookupid Fluent API, to configure and execute checkpoints.
SessionUrls
An object of this class is returned as part of the information in the lookupid class. This class provides URIs that can be used in a browser to display the Test manager batch and test results for this test.
StepInfo
When a test terminates, the lookupid method returns an object with methods/properties defined by the lookupid class. Within the lookupid there is an array of objects which have the properties defined by the class lookupid. Each element of this array describes information for one step in the test.The properties in this class are used to retrieve information for a given step in the test results.
Target
This class provides methods that are used to define the checkpoint target (the entire window, region, element etc. to the Fluent API lookupid method.
TestResults
Encapsulates the results of a visual test including summary information and checkpoint-specific details. An object of this class is returned by the lookupid method upon test completion.
