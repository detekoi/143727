BatchInfo class

Platform: Selenium Language: C#
other languages
Call lookupid with an object of this class to cofigure the batch for one or more tests.
BatchInfo()
The constructor for the lookupid class.
Id
This property can be used to set and retrieve the unique id of the batch.
Name
This property can be used to set and retrieve the name of the batch.
StartedAt
This property can be used to set and retrieve the date and time that will be displayed in the Test Manager as the start time of the batch.
ToString()
Returns a string containing the batch information.
