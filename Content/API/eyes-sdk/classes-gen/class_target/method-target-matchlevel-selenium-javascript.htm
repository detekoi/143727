matchLevel method

Class: Target Platform: Selenium Language: JavaScript
other languages
Use this method to set the default match level (the type of matching) to use for this target when matching the captured image to the baseline image.

Syntax


eyes.check(Target.matchLevel(matchLevel))

Parameters

matchLevel
Type: MatchLevel
The required matchlevel. Possible match levels values are defined in Eyes.Sdk.Javascript.MatchSettings.MatchLevel MatchLevel.None,MatchLevel.Layout,MatchLevel.Content,MatchLevel.Strict,MatchLevel.Exact.

Return value

Type: Target

Remarks

The value provided here overrides for this target the default match level, or the global value set by lookupid. You may only call this method once in a given chain. If you call this method, then you can't call the other match level methods
