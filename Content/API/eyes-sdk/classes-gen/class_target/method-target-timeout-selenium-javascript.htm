timeout method

Class: Target Platform: Selenium Language: JavaScript
other languages
Defines the timeout to use when acquiring and comparing screenshots for this target.

Syntax


eyes.check(Target.timeout(ms))

Parameters

ms
Type: number
The timeout to use in milliseconds.

Return value

Type: Target

Remarks

You may only call this method once in a given chain. The timeout value provided here overrides for the current target, the default value, or the value set by lookupid. Passing this method a value of zero disables the timeout mechanism.
