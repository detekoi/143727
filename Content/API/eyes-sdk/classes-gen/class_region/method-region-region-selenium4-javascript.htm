Region method

Class: Region Platform: Selenium 4 Language: JavaScript
other languages
A constructor that provides several ways to provide the region rectange size and coordinates.
For user conveniance the values may be given as numbers or taken from another object.

Syntax


let region = new Region(region)
let region = new Region({left: number, top: number, width: number, height: number})
let region = new Region(location, size)
let region = new Region(left, top, width, height)

Parameters

left
Type: number
The X coordinate of the left edge of the region in pixels.
top
Type: number
The Y coordinate of the top edge of the region in pixels.
width
Type: number
The width of the region in pixels.
height
Type: number
The height of the region in pixels.
region
Type: Region|RegionObject
The region provided as a parameter specifies both the top left of the new region and its dimensions and optionally the type of the coordinates. Instead of an object of the Region class you can provide any object that meets the following definition: 
 {left: number, top: number, width: number, height: number, coordinatesType: ?CoordinatesType}
location
Type: Location
The object passed as a parameter specifies the top left of the new region. Instead of an object of the Location class you can provide any object that meets the following definition: 
 {x: number, y: number}
size
Type: RectangleSize
The object provided as a parameter specifies dimensions of the region. Instead of an object of the Location class you can provide any object that meets the following definition: 
 {width: number, height: number}

Return value

Type: Region

Remarks

None.
