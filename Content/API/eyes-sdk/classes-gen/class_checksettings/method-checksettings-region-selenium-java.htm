region method

Class: SeleniumCheckSettings Platform: Selenium Language: Java
other languages
Requests that only a region within a selected i-frame should be matched.

Syntax


eyes.check(Target.frame(frameSelector).region(region))
eyes.check(Target.frame(frameSelector).region(by))

Parameters

region
Type: Region
An object that defines a rectangular area in the window.
by
Type: By
An element selector which is chosen using the Selenium By class.

Return value

Type: SeleniumCheckSettings

Remarks

The parameter to the method specififes the region to be matched. The enclosing frame(s) are selected by the previous calls in the chain to lookupid You may only use this method in the chain if the first method in the chain is lookupid. You may not call it if the first method of the chain is lookupid or lookupid You may only call this method once in a given chain. To match a region within a window use the method lookupid as the first methof of the Target chain.
