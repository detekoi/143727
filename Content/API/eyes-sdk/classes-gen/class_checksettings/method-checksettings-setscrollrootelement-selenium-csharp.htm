ScrollRootElement method

Class: SeleniumCheckSettings Platform: Selenium Language: C#
other languages
Normally, Eyes will select the most appropriate element to scroll to execute the lookupid method. This method is available to allow the user to specify the element to scroll explicitly.

Syntax


eyes.Check(Target.Window().ScrollRootElement(selector))
eyes.Check(Target.Window().ScrollRootElement(element))

Parameters

selector
Type: By
A selector to the element that should be scrolled.
element
Type: IWebElement
The element that should be scrolled.

Return value

Type: SeleniumCheckSettings

Remarks

None.
