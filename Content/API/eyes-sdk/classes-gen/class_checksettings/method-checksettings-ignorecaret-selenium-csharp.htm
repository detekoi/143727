IgnoreCaret method

Class: SeleniumCheckSettings Platform: Selenium Language: C#
other languages
Use this method to tell Eyes that for this target it should detect mismatch artifacts caused by a blinking cursor and not report them as mismatches.

Syntax


eyes.Check(Target.Window().IgnoreCaret(ignoreCaret))
eyes.Check(Target.Window().IgnoreCaret())
eyes.Check(Target.Window().IgnoreCaret(ignoreCaret))
eyes.Check(Target.Window().IgnoreCaret())

Parameters

ignoreCaret
Type: bool [Optional : default = true ]
If this parameter has a value of true, then Eyes will detect artifacts caused by a blinking cursor and will not report a mismatch. If it has a value false, then no blinking cursor detection is done. If no parameter is passed then the default value is true - so the cursor artifacts will be ignored.

Return value

Type: ICheckSettings

Remarks

You may only call this method once in a given chain. This method overrides for the current target, the global default setting or the setting set explicitly using the lookupid.
