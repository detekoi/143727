Select a frame that is direct descendent of a frame selected previously in the chain.

Syntax


eyes.check(Target.frame(frame).frame(frame))

Parameters

frame
Type: Integer|string|By|WebElement

Return value

Type: SeleniumCheckSettings

Remarks

You may only use this method in the chain if the first method in the chain is lookupid. You may not call it if the first method of the chain is lookupid or lookupid call this method multiple times to descend a hierarchy of embedded frames.
