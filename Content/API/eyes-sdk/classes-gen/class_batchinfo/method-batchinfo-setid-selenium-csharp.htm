Id property

Class: BatchInfo Platform: Selenium Language: C#
other languages
This property can be used to set and retrieve the unique id of the batch.

Syntax


string value; // give relevant initial value
obj.Id  = value;
value = obj.Id

Property Type

string

Remarks

If the id is not set, and the environment variable APPLITOOLS_BATCH_ID is defined, then its value will be used as the id. If the environment variable is not defined, then an internal unique id will be generated and used.
