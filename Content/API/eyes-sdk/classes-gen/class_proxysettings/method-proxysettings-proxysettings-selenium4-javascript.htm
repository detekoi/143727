ProxySettings method

Class: ProxySettings Platform: Selenium 4 Language: JavaScript
other languages
This is the constructor for the lookupid class.
To initiate the use of a proxy server, create a lookupid object with this constructor and then pass the object to the lookupid method.

Syntax


value = proxySettings.ProxySettings(uri, username, password)

Parameters

uri
Type: string
The proxy's server URI.
username
Type: string [Optional ]
The username to be sent to the proxy (default none).
password
Type: string [Optional ]
he password to be sent to the proxy (default none).

Return value

Type: ProxySettings

Remarks

None.
