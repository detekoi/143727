setViewportSize method

Class: Eyes Platform: Selenium 4 Language: JavaScript
other languages
Use this method to set the viewport size directly (using the driver). Call this method if you want to set the window size before the call to open. Then, when calling the open method, don't pass the viewportSize parameter.

Syntax


eyes.setViewportSize(driver, viewportSize)

Parameters

driver
Type: EyesWebDriver
The driver returned from the call to open.
viewportSize
Type: RectangleSize

Return value

Type: Promise<void>

Remarks

None.
