abortIfNotClosed method

Class: Eyes Platform: Selenium Language: JavaScript
other languages
Call this in an exception handler that encloses the entire test to catch cases where lookupid was called, an exception is thrown, and execution stops without calling lookupid.
This method does all the cleanup normally done by lookupid. If this method is called, and lookupid has not been called, then the test will have a status of Aborted in the Test Manager.

Syntax


eyes.abortIfNotClosed()

Parameters

This method does not take any parameters.

Return value

Type: This method does not return a value.

Remarks

None.
