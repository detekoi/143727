MatchTimeout property

Class: Eyes Platform: Selenium Language: C#
other languages
Use this property to retrieve and set the maximum time Eyes will try to perform a match on the fully captured image.
If there is a mismatch then Eyes will recapture and recheck the checkpoint until the timeout limit. This can help avoid mismatches in cases where the capture was done before the page was fully rendered. The default timeout is set by the lookupid.

Syntax


TimeSpan value; // give relevant initial value
eyes.MatchTimeout  = value;
value = eyes.MatchTimeout

Property Type

TimeSpan

Remarks

None.
