setMatchLevel method

Class: Eyes Platform: Selenium Language: JavaScript
other languages
Use this method to set the default match level to be used for tests.
You can set a specific match level for a particular step, overiding the default set by this method, using the check fluent interface method lookupid.

Syntax


eyes.setMatchLevel(level)

Parameters

level
Type:
The required match level.

Return value

Type: This method does not return a value.

Remarks

Possible match levels values are:
  • MatchLevel.None
  • MatchLevel.Strict
  • MatchLevel.Layout
  • MatchLevel.Content
  • MatchLevel.Exact
