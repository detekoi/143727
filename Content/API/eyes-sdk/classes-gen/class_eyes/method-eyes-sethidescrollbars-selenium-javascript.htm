setHideScrollbars method

Class: Eyes Platform: Selenium Language: JavaScript
other languages
Use this method to set whether or not Eyes hides the scrollbars before capturing screenshots.

Syntax


eyes.setHideScrollbars(hide)

Parameters

hide
Type: boolean
If set to true then hide the scrollbars. If set to false don't hide the scrollbars.

Return value

Type: This method does not return a value.

Remarks

None.
