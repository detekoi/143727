Parameters

selector
Type: By
An object that defines a selector for the region.
tag
Type: string
stitchContent
Type: bool
If true, then stitch the content if the application window is larger than the viewport, if false don't stich.
matchTimeout
Type: int [Optional : default = -1 ]
If there is a mismatch then Eyes will recapture and recheck the checkpoint until the timeout limit. This can help avoid mismatches in cases where the capture was done before the page was fully rendered. This parameter overides any global timeout set using lookupid. The retry time, the units are defined by the TimeSpan type.
matchTimeout
Type: TimeSpan
The timeout in TimeSpan units.
region
Type: Rectangle
A nonempty region representing the screen region to check in screenshot pixel coordinates.

Return value

Type: void
