getIsOpen method

Class: Eyes Platform: Selenium 4 Language: JavaScript
other languages
Use this to know if the Eyes instance is currently open

Syntax


value = eyes.getIsOpen()

Parameters

This method does not take any parameters.

Return value

Type: boolean
return lookupid if lookupid has been called on an eyes instance, and neither lookupid nor lookupid have been called subsequently.
