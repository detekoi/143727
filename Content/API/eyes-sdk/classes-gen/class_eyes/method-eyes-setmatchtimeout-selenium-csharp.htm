MatchTimeout property

Class: Eyes Platform: Selenium Language: C#
other languages
This property can be used to set and retrieve the maximum time Eyes will try to perform a match on the fully captured image.
Since a browser can take time to render a page (because it is complex, or because of slow network speeds) if Eyes detects mismatches, it will initially assume that the mismatch was because the render has not completed yet, and it will retry the match after a short wait. You can use this property to determine how much time Eyes continue to retry the matching before declaring a mismatch. You can overide the default timeout set by this property for a specific step by using the lookupid in a fluent lookupid chain. Various other check method variations also take a timeout parameter for the same purpose.

Syntax


TimeSpan value; // give relevant initial value
eyes.MatchTimeout  = value;
value = eyes.MatchTimeout

Property Type

TimeSpan
