checkRegionByElement method

Class: Eyes Platform: Selenium 4 Language: JavaScript
other languages
Run a checkpoint only on a specific region defined by an element.

Syntax


promise = eyes.checkRegionByElement(element, tag, matchTimeout)

Parameters

element
Type: WebElement
An object whose area defines the area for the region.
tag
Type: string
matchTimeout
Type: number
If there is a mismatch then Eyes will recapture and recheck the checkpoint until the timeout limit. This can help avoid mismatches in cases where the capture was done before the page was fully rendered. This parameter overides any global timeout set using lookupid. The retry time in milliseconds.

Return value

Type: Promise<MatchResult>

Remarks

None.
