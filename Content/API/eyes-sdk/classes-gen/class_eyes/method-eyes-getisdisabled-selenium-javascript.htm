getIsDisabled method

Class: Eyes Platform: Selenium Language: JavaScript
other languages
Use this method to retrieve whether or not interactions with Eyes is being ignored.
You can enable and disable Eyes using the lookupid.

Syntax


value = eyes.getIsDisabled()

Parameters

This method does not take any parameters.

Return value

Type: boolean
If true, then all calls to Eyes are being ignored, if false then all calls to Eyes are working as usual.

Remarks

None.
