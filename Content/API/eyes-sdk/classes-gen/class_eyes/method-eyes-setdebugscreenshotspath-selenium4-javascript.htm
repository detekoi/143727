setDebugScreenshotsPath method

Class: Eyes Platform: Selenium 4 Language: JavaScript
other languages
Use this method to set that folder path where locally saved checkpoint images should be stored (if enabled).

Syntax


eyes.setDebugScreenshotsPath(pathToSave)

Parameters

pathToSave
Type: string
A pathname where saved checkpoint images should be stored.

Return value

Type: This method does not return a value.

Remarks

Enable and disable saving screenshots to the local disk using the lookupid.
