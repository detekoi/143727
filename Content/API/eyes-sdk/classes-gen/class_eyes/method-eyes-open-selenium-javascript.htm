open method

Class: Eyes Platform: Selenium Language: JavaScript
other languages
Call this method to start a test, before calling any of the check methods.

Syntax


promise = eyes.open(appName, testName, viewportSize)
promise = eyes.open(driver, appName, testName, viewportSize)

Parameters

appName
Type: string
testName
Type: string
The name of the test. This name must be unique within the scope of the application name. It may be any string.
viewportSize
Type: {width: number, height: number}
driver
Type: WebDriver
The web driver.

Return value

Type: Promise<WebDriver>
