setProxy method

Class: Eyes Platform: Selenium Language: Java
other languages
Use this method to configure Eyes so that it will interact with the Eyes server via a proxy.
By default the SDK interacts directly with the Eyes server. If necessary, you can define a proxy server so that the SDK will interact with the proxy server, and the proxy server will relay the interaction with the Eyes server. For more details see Configuring a proxy server.

Syntax


eyes.setProxy(proxySettings)

Parameters

proxySettings
Type: ProxySettings
An object created from the lookupid class that contains the user name, password, URI and port

Return value

Type: void

Remarks

None.
