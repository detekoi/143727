MatchLevel property

Class: Eyes Platform: Selenium Language: C#
other languages
This property can be used to set and retrieve the default match level to be used for tests.
You can set a specific match level for a particular step, overiding the default set by this method, using the check fluent interface method lookupid. You can also define that specific regions in a target have a particular match level using the methods lookupid, lookupid, lookupid.

Syntax


MatchLevel value; // give relevant initial value
eyes.MatchLevel  = value;
value = eyes.MatchLevel

Property Type

MatchLevel

Remarks

Possible match levels values are:
  • MatchLevel.None
  • MatchLevel.Strict
  • MatchLevel.Layout
  • MatchLevel.Content
  • MatchLevel.Exact
