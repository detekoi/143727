RectangleSize method

Class: RectangleSize Platform: Selenium 4 Language: JavaScript
other languages
A constructor that provides several ways to provide the values for the rectangle width and height.

Syntax


let rect = new RectangleSize(width, height)
let rect = new RectangleSize(anotherRectangleSize)
let rect = new RectangleSize({width: number, height: number})

Parameters

arg1
Type: number|RectangleSize|RectangleSizeObject
If this argument is a number, then it defines the width of the rectangle in pixels and the second parameter defines the width. If this argument is an object, then both the width and height are taken from the object. The RectangleSizeObject may be any type of object that has the width and height properties.
arg2
Type: number [Optional ]
If the first parameter is a number, then it defines the width of the rectangle in pixels and this parameter defines the width. Otherwise this parameter is ignored.

Return value

Type: RectangleSize

Remarks

None.
