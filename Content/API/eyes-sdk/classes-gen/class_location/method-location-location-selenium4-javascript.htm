Location method

Class: Location Platform: Selenium 4 Language: JavaScript
other languages
A constructor that provides several ways to provide the Location X and Y coordinates.
For user conveniance the values may be given as numbers or taken from another object.

Syntax


let location = new Location(x, y)
let location = new Location(otherLocation)
let location = new Location({x: number, y: number})

Parameters

arg1
Type: number|Location|LocationObject
If this parameter is a number then it is the X coordinate and the second parameter is the Y coordinate. Otherwise, this parameter should either be another Location object in which case a clone is returned or it should be any other object that has properties called x and y that are numbers, in which case these values are used to clone a new Location object.
arg2
Type: number [Optional ]
If the first parameter is a number then this parameter is the Y coordinate, otherwise this parameter is ignored

Return value

Type: Location

Remarks

None.
