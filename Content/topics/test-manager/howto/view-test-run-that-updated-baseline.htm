View the results of the test run which updated a baseline

You can display the results of the test run that most recent updated a particular baseline from various pages in the . On all the pages you do this using the row menu on the list of items presented on that page. See below for the list of pages and which menu entry you use:

page menu item Selected run
Apps & tests Show last save Show the last test run that updated any of the baselines of the selected test.
Test baselines Show saved run Show the last test run that updated the selected baseline.
Branch baselines Show saved run Show the last test run that updated the selected baseline.
Baseline history Show saved test Show the test run that when saved, created the selected baseline revision.

If the results of a test run is not available, for example, because it has been deleted, then you will see a warning dialog, and then the test results page will be displayed showing the run of the most recent batch.
