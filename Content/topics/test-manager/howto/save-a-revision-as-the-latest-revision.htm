Save a revision as the latest revision

Navigate to the The baseline history page page from the The branch baselines page or The apps & tests page. See Navigate to baseline history page for detailed instructions on how to do this.

Find the row with the revision you are interested in, and on the row menu select Save as latest. This entry will be grayed out if the revision already is the latest revision. The information in the selected revision will be cloned, and a new revision will be stored with the current time and date. Any tests now run on this baseline will be compared to the newly cloned baseline.
