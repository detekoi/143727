How to delete test runs

This article describes how you can delete from the server test results you no longer require . Note that removing a test result does not delete the baseline used or updated by that results. Removing the results does not impact future runs of the test in any way.

You can delete test results that from two places in the
  • On the Test results page, you can delete either an entire batch  or a subset of the tests of a batch.
  • On the Last test run, page you can delete test runs for a particular test (application/test name pair) irrespective of when they were run or under which batch or execution environment.
On both pages, the deletion method is identical.
