Pages and editors overview

provides multiple pages, each of which presents a different view of your test and baseline information, and allows you to perform different tasks. This article presents an overview of what you can achieve on each page, and how to access that page.

You access some pages using the Page navigator, other pages are reachable from within the context of another page. When closed, the Page navigator always displays the name of the currently displayed page.

The following table summarizes the available pages, their main features and how to navigate to them. Click on the page name to see more information about that page.
