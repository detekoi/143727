The baseline details view

The branch baseline and test baseline pages provides tools in the toolbar that allow you to choose from three different views:

This article describes the baseline details view.

Page layout

When you first enter this page make sure that the branch is set to the branch you are interested in. See Selecting a branch on a page for more details.

In this view, the baselines in the branch are laid out as a table with one row per baseline:

The row for each baseline includes the following columns of information:

Name
The test name
App
The application name.
OS
The operating system the test that created the baseline ran on.
Browser
The browser the test that created the baseline displayed on (for web-based systems).
Viewport
The viewport size that was active when the test that created the baseline started.
Saved
The date and time when the baseline was most recent saved.
Saved by
The name of the team member that did the most recent save to the baseline (empty if the baseline was created and saved automatically).
Test runs
The number of test runs executed using this baseline (this will have a value of 0 if the baseline was only created and has not yet been used as a reference for another test).

Each line in the baseline has two controls:

  • Use this checkbox, to select a baseline , and then :
    • Click in the toolbar to copy the selected baselines to another branch. (See Copying baselines between branches.)
    • Click in the toolbar to delete the selected baselines. (See Deleting baselines.)

    You can also use the toolbox filter and checkbox controls to select all rows that meet some criteria.

  • The menu button appears at the right end of a row when you hover the cursor over that row. When clicked, the menu offers the following options:
    Show last run
    Display the Test results page for the test run that ran most recently against this baseline. This option will be grayed out if a test has not run against this baseline. See View the results of the latest run of a baseline
    Show saved run
    Display the test result page for the test run that most recently updated this baseline. See View a test run which updated a baseline results.
    Show history
    Display the Baseline history page with a list of the revisions of this baseline. See The Baseline history page.
    Delete
    Delete this baseline. See Deleting Baselines.

If there are many baselines, then you can use the following tools to adjust what baselines you see and how they are ordered:

  1. Use the filter tool in the toolbar, to limit the baselines that you see, to those that meet the criteria you specify.
  2. Sort the rows by any column by clicking on the column header.
  3. Organize the baselines into a hierarchy by using the .

See Filtering and grouping in baseline views for more information on these options.

Step thumbnail features

By default when you open this page, it shows one row of information for each baseline. If you click on any baseline row, then that row will expand to display a thumbnail for each step in the baseline.

The difference between this view (baseline details view with displayed thumbnails), and the baseline step view is that in this view the steps are always grouped by test. You can use sorting by any column to compare the images by another criteria. In the step view all the steps are shown, and you can organize them into any convenient hierarchy, by any of the supported criteria, irrespective of the what test they ran in.
