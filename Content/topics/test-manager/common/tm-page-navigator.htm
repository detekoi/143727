The page navigator

The Page navigator is a control at the top of the window, that is used to switch between the main pages of the Test manager. It also displays the name of the current page or editor.

Note that same pages and editors whose name appear in the page navigator when they are active, are not directly accessible from the page navigator, but are accessed from other pages by clicking on an item or from an item menu.
