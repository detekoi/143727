How to create a New Bug or Issue

To report a new bug do as follows :

  1. click the new issue button (i)
  2. mark a rectangular area to indicate the area on the screen you are commenting on
  3. enter a one line title for the issue
  4. press create or press enter
  5. After this will open a dialog that allows you various options such as
    • add a description of the bug
    • Add the name of other team members so that they will also be notified of the problem
    • define how to report this mismatch in future runs

Accepting Baseline Images for Bugs

Once you have reported that a mismatch is a bug, you need to decide what to do with this mismatch in future runs. On the one hand you don't want to report this bug more than once, on the other hand, you do want the diff to be noted until the bug is fixed. A related question is if you should reject or accept the new actual image as the baseline image. If you accept it then on subsequent runs, as long as the bug is not fixed, it won't be detected. If you reject it then it will be reported as a new bug every time.
