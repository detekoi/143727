Delete an app and its tests

Navigate to the "Apps & tests" page. In the left panel, when you hover your mouse over any application name, a menu will show up. Click on the menu and select "delete". This will delete the application name and all the associated test names. It does not delete the test results or baselines associated with the tests in the deleted application.

Delete multiple applications

Navigate to the "Apps & tests" page. In the left panel, click the checkboxes of the applications you want to delete. Then click the delete application button . will display a confirmation dialog, click "Confirm" to proceed with the deletion or "Cancel" to abort the deletion request. You can delete all the applications by clicking the checkbox in the header at the top of the list of apps, or by selecting "All apps" in the click down menu to the right of the checkbox. Finding an application name

To quickly find an application, or a set of applications with common text in their name, type in the "Filter apps" control, as you type, the list of apps will be filtered so that only apps that contain the text you typed will appear. Tip: you can now used the checkbox at the top of the list of apps to select those applications.
