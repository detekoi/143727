The step thumbnail tool

When the screen is in one of the test and step views (Test details, Batch steps by test, Batch steps) each step is represented by a step thumbnail.

The step thumbnail displays information about the step and also provides various operations.

The following information is displayed

  • It displays the checkpoint image or the baseline image (switchable)
  • It displays any differences that were found
  • It displays the match status (match, mismatch, missing baseline, missing checkpoint)
  • It displays if the user has done accept or reject
  • Displays if the user has entered an issue
  • The user can accept or reject a step
