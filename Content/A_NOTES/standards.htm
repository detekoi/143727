standards

This internal document defines the documentation standards.

SDK

If you want a phrase to be SDK language dependent then put it in a spa as follows

<span data-mpld-lang="java php"</span>

If you want to refer to a symbol (method or property) in a language independent way then refer to it as follows:

<span data-mpld-lookup="setappname">setappname</span>

<span data-mpld-lookup methodOrProp = "setappname">method</span>

The first will be replace by the appropriate name according to the current language, the second will be replaced by the word method or property as appropriate.

TBD in the future there will be an option to define if a cross reference link should be added or not.
