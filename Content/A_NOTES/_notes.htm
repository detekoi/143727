Some more setup

There are some other setup functions that you might want to consider adding to your tests:

eyes.setHideScrollbars(true);. If you find that finds mismatches because the scroll bar in your test changes between runs, and this is not considered an issue, then you can suppress the scroll bars and prevent the mismatch from being reported.

eyes.setForceFullPageScreenshot(true). In some browsers, if the viewport is smaller than the window content, then taking a screenshot will only capture the visible portion of the window. If you call this method, then will use scrolling to automatically take multiple screenshots and to stitch them together into a single image of the entire window content.
